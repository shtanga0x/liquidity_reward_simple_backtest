Reward Formula (Polymarket)

Order score: S(v, s) = ((v − s) / v)² × size

v = max spread (¢ / 100),  s = |your_price − midpoint|,  size = dollars

Two-sided scoring (c = 3):

Qyes = YES buy scores  |  Qno = NO buy scores

Qeff = max( min(Qone, Qtwo), max(Qone/c, Qtwo/c) )

Extreme midpoints (0–0.10 or 0.90–1.0):

Qeff = min(Qone, Qtwo)  — two-sided orders required

Daily reward = (your Qeff / total market Qeff) × daily pool
